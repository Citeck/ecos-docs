API
====

Мобильное приложение Citeck взаимодействует с серверной частью платформы через
набор REST API. API обеспечивает получение списков задач, разбор форм для
отображения в мобильном интерфейсе, применение фильтров к данным, а также работу
с вкладками задач и профиля пользователя. В данном разделе приведена документация
по каждому из доступных API-методов мобильного приложения.
