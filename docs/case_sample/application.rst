.. _app_example:

Упаковка приложения
======================

.. contents::
   :depth: 2

Созданные артефакты можно упаковать в виде приложения — например, чтобы поместить в систему контроля версий или перенести на другой стенд.

**Приложение** — это zip-архив, содержащий артефакты системы: типы данных, формы, журналы, бизнес-процессы.

Создание приложения
--------------------

Перейдите в рабочее пространство администратора. В разделе **Управление системой** выберите пункт **Приложения ECOS**:

.. image:: _static/app/App_1.png
   :width: 700
   :align: center

|

Нажмите **+ — Создать приложение**:

.. image:: _static/app/App_2.png
   :width: 500
   :align: center

|

.. image:: _static/app/App_3.png
   :width: 600
   :align: center

|

В открывшейся форме заполните поля:

1. **Id** — идентификатор приложения.
2. **Имя** — название приложения.
3. Выберите все созданные **типы данных** :ref:`для процесса «Заявка на закупку оборудования»<sample_request>`.
4. Нажмите **«Загрузить артефакты для выбранных типов»** — список артефактов заполнится автоматически.

При необходимости дополните список вручную, нажав **«Добавить»**.

.. image:: _static/app/App_4.png
   :width: 600
   :align: center

|

Нажмите **Сохранить**.

Скачивание приложения
-----------------------

Чтобы скачать архив с приложением, нажмите кнопку скачивания напротив нужного приложения:

.. image:: _static/app/App_5.png
   :width: 700
   :align: center

Загрузка на другой стенд
--------------------------

Чтобы загрузить приложение на другом стенде, нажмите **Загрузить**:
