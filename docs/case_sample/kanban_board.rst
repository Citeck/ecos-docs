Пример добавления канбан-доски
================================

.. note::

   Данная статья является продолжением работы с бизнес-процессом :ref:`Заявка на закупку оборудования<sample_request>`.

.. tip::

   Для создания канбан-доски в локальном рабочем пространстве перейдите в **Администрирование → Канбан-доски**.

Канбан-доска позволяет визуализировать работу с заявками по статусам в удобном формате.

.. seealso::

   Подробнее о возможностях инструмента см. :ref:`Канбан-доска<kanban_board>`.

Создание конфигурации
----------------------

Перейдите в рабочее пространство администратора: **Конфигурация UI → Канбан-доски**.

.. image:: _static/kanban/01.png
   :width: 700
   :align: center

Нажмите **«+ Создать конфигурацию канбан-доски»** и заполните поля:

.. list-table::
   :widths: 5 25 40
   :header-rows: 1
   :align: center
   :class: tight-table

   * - №
     - Поле
     - Значение
   * - 1
     - Идентификатор доски
     - ``kanban_equipment_request``
   * - 2
     - Имя канбан-доски
     - Канбан — Заявка на покупку оборудования
   * - 3
     - Тип данных
     - ранее созданный тип «Заявка на покупку оборудования»
   * - 4
     - Журнал
     - ранее созданный журнал «Заявка на покупку оборудования»
   * - 5
     - Колонки
     - статусы из типа данных (доступные для выбора отображаются в списке автоматически)

Итоговая конфигурация:

.. image:: _static/kanban/02.png
   :width: 600
   :align: center

Просмотр канбан-доски
----------------------

Перейдите в журнал **«Заявка на покупку оборудования»** и переключитесь на представление канбан-доски:

.. image:: _static/kanban/03.png
   :width: 700
   :align: center

Карточки заявок будут автоматически распределены по столбцам в соответствии с их текущим статусом.
