Архитектура
===================

Платформа Citeck построена на микросервисной архитектуре, что обеспечивает горизонтальное масштабирование, независимое развёртывание компонентов и гибкость при интеграции с внешними системами. Каждый микросервис отвечает за свою предметную
область: управление моделью данных, бизнес-процессами, уведомлениями, трансформацией контента и другими функциями. Взаимодействие между микросервисами осуществляется через брокер сообщений (RabbitMQ) и REST API. 

В данном разделе рассматриваются базовые принципы архитектуры и механизм патчей для расширения функциональности.
