.. _launcher_update:

Обновление Citeck Community
-------------------------------------------

1. Создайте :ref:`снапшот данных <launcher_dump>`, чтобы при необходимости можно было восстановить исходное состояние.

2. Перейдите в **режим редактирования** namespace:

   .. image:: _static/update_01.png
       :width: 600
       :align: center

3. Выберите **бандл** нового релиза и нажмите **Confirm**:

   .. image:: _static/update_02.png
       :width: 600
       :align: center

4. Нажмите кнопку запуска:

   .. image:: _static/update_03.png
       :width: 600
       :align: center

5. Нажмите **Update & Start**:

   .. image:: _static/update_04.png
       :width: 600
       :align: center

6. Начнётся скачивание и разворачивание образов:

   .. image:: _static/update_05.png
       :width: 600
       :align: center

7. Дождитесь статуса **Running** всех микросервисов и приложений, затем нажмите **Open In Browser**.

.. note::

    Данные не удаляются автоматически при обновлении — для их удаления требуется явное действие.
