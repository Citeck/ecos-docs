Работа с секретами и мастер-паролем
--------------------------------------------------------------------

**Мастер-пароль** шифрует все секреты, используемые в лончере. Его потребуется вводить один раз при каждом перезапуске лончера.

Для перехода к управлению секретами нажмите соответствующую кнопку:

.. image:: _static/secret_0.png
    :width: 700
    :align: center


Установка мастер-пароля
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Введите и подтвердите мастер-пароль:

.. image:: _static/secret_1.png
    :width: 400
    :align: center


Просмотр и удаление секретов
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Для просмотра списка секретов введите мастер-пароль:

.. image:: _static/secret_4.png
    :width: 400
    :align: center

Откроется список секретов с возможностью удаления каждого из них:

.. image:: _static/secret_2.png
    :width: 400
    :align: center


Сброс мастер-пароля
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. note::

    При сбросе все сохранённые секреты будут удалены.

Нажмите кнопку сброса и подтвердите действие:

.. image:: _static/reset_secret_1.png
    :width: 400
    :align: center

.. image:: _static/reset_secret_2.png
    :width: 400
    :align: center

Затем задайте новый мастер-пароль:

.. image:: _static/reset_secret_3.png
    :width: 400
    :align: center
