Пример настройки процесса
==========================

В разделе описан пример настройки типового процесса HR-отдела по согласованию офферов для кандидатов-соискателей.

В ECOS процесс включает в себя создание **справочника «Кандидаты»** и **кейса «Согласование оффера»**.

И для справочника, и для кейса в системе последовательно создаются следующие артефакты, связанные между собой:

-	**Тип данных** — характеристика набора данных, которая определяет его возможные значения, список допустимых действий, связей, прав доступа а также способ хранения набора данных.
-	**Форма** – графическое представление объекта в виде набора полей, соответствующих атрибутам объекта. 
-	**Журнал** - табличная форма представления данных с возможностью настройки отображения столбцов, где используются те же атрибуты, что и в формах.
