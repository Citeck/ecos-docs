Документация Citeck
====================

.. image:: _static/Logo_Citeck_SL.svg
      :width: 300
      :target: https://www.citeck.ru/


С использованием **Citeck** могут быть реализованы:

    * управление бизнес-процессами, используя встроенный конструктор BPMN;
    * портальные решения;
    * приложения для работы с корпоративной информацией;
    * ECM-решения, в том числе электронные архивы;
    * CRM-системы.

**Citeck** сочетает в себе:

    * **No-Code** - создание бизнес-приложений без навыков программирования с использованием :ref:`веб-редактора артефактов и BPMN-процессов<no_code_constructor>`.
    * **Low-Code** - добавление более сложной бизнес-логики для приложений, созданных с помощью No-Code и BPMS и программирование в визуальном интерфейсе.
    * **Pro-Code** - реализация сложной бизнес-логики в :ref:`собственном микросервисе<mcs_setup>`. Написание кода на обычных или специализированных языках программирования, с использованием сред разработки на этих языках.

Версия **Citeck Community** – решение с открытым исходным кодом (open source), распространяемое по лицензии `LGPL <https://www.gnu.org/licenses/lgpl-3.0.ru.html>`_, которое содержит базовый функционал для управления корпоративным контентом (ECM) и бизнес-процессами (BPM). 

В статьях представлена информация по установке Citeck Community:

    * :ref:`Развертывание Citeck с использованием кросплатформенного лончера <quick_start>`; 
    * :ref:`Установка Citeck c помощью Docker Compose<docker_compose>` 
    * :ref:`Развертывание виртуальной машины с Citeck в VirtualBOX <virtualbox>`; 

В разделе :ref:`"Знакомство с Citeck"<intro>` описан интерфейс и порядок работы с готовыми модулями системы.

В разделе :ref:`"Пример настройки процесса"<sample>` размещены примеры, описывающие создание процесса:

    * пример начального уровня для ознакомления с возможностями Citeck - автоматизация :ref:`процесса закупки оборудования<sample_request>`; 
    * более сложный и объемный пример для автоматизации типового процесса HR-отдела :ref:`по согласованию офферов для кандидатов-соискателей<sample_offer>`
