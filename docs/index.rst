Документация Citeck ECOS 
=========================

С использованием **Citeck ECOS** могут быть реализованы:

    * управление бизнес-процессами, используя встроенный конструктор BPMN;
    * портальные решения;
    * приложения для работы с корпоративной информацией;
    * ECM-решения, в том числе электронные архивы;
    * CRM-системы.

Версия **Citeck ECOS Community** – решение с открытым исходным кодом (open source), распространяемое по лицензии LGPL, которое содержит базовый функционал для управления корпоративным контентом (ECM) и бизнес-процессами (BPM). 

В :ref:`статье<docker_compose>` представлена информация по установке Citeck ECOS Community.

В разделе :ref:`"Знакомство с ECOS"<intro>` описан интерфейс и работы с готовыми модулями системы.

В разделе :ref:`"Пример настройки процесса"<sample>` размещены примеры, описывающие создание процесса:

    * пример начального уровня для ознакомления с возможностями ECOS - автоматизация :ref:`процесса закупки оборудования<sample_request>`; 

    * более сложный и объемный пример для автоматизации типового процесса HR-отдела :ref:`по согласованию офферов для кандидатов-соискателей<sample_offer>`

Далее представлены разделы базы знаний, в статьях которой подробнее описана работа со сложными и неочевидными настройками, возможностями изменения и персонализации системы.
