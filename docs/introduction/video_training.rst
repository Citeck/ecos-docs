Видео по работе с системой
==============================

Предлагаем Вашему вниманию записи технологических тренингов по работе в Low-Code платформе Citeck:

1 день. Знакомство с системой, функциональностями ECM
----------------------------------------------------------

- Знакомство с интерфейсом: рабочая область, меню, раздел администратора
- Создание бизнес-приложения "Заявка Service Desk" в 5 шагов: тип данных, атрибуты, роли, статусы и BPMN-процесс. Запуск и проверка его работоспособности.
- Обзор возможности переноса артефактов, упаковки бизнес-приложения, выгрузки/загрузки бизнес-приложения, интеграции с Git
- Добавление вычисляемых полей в тип данных
- Изменение журнала и формы, которые были созданы по умолчанию из типа данных
- Обзор представления данных: журнал, канбан-доска, библиотека документов
- Настройка дашбордов и виджетов
- Настройка меню
- Настройка действий

2 день. Бизнес-процесс
-----------------------------

- Усложнение бизнес-процесса: добавление ветвления, уведомления, событий (таймер, сигнал), событийный подпроцесс
- Встраивание модели принятия решений (DMN) в бизнес-процесс
- Настройка уведомлений, lazy approval, массовой рассылки
- Настройка KPI
- Обзор возможностей виджета «Статистика процесса» на основе созданного бизнес-процесса
- Администрирование созданного бизнес-процесса - миграция версий, миграция токенов
- Настройка виджета «Графическая статистика»
