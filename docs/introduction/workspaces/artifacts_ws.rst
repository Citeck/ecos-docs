Создание артефактов в рабочем пространстве
----------------------------------------------------------------------------------

.. _ws_artifacts:

Базовые no-code/low-code инструменты разработки доступны непосредственно внутри совместного рабочего пространства — без необходимости переходить в пространство администратора.

Пользователь с ролью **Менеджер** может создавать :ref:`артефакты<ecos_artifacts>`. Их перечень представлен в разделе **Администрирование** данного пространства:

 .. image:: _static/ws_artifacts_01.png
       :width: 700
       :align: center

.. note::

  Все артефакты, созданные в рамках рабочего пространства, остаются доступны только в том пространстве, где они были созданы. При необходимости использования артефакта в нескольких пространствах следует создавать его через пространство администратора.
