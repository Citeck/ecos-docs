Создание артефактов в рабочем пространстве
--------------------------------------------

.. _ws_artifacts:

Базовые no-code/low-code инструменты доступны внутри созданного рабочего пространства.

Пользователь с ролью **Менеджер** может создавать :ref:`артефакты<ecos_artifacts>`, перечень которых представлен в разделе **Администрирование** этого пространства.

 .. image:: _static/ws_artifacts_01.png
       :width: 700
       :align: center 

.. note::

  Все артефакты, созданные таким образом, остаются доступны только в том рабочем пространстве, где они были созданы.

.. list-table::
      :widths: 10 30
      :align: center

      * - :ref:`Типы данных<data_types_main>`
        - Пример создания :ref:`типа данных "Заявка на покупку оборудования"<ws_artifacts_data_type>`.
      * - :ref:`Шаблоны нумерации<number_template>`
        - Пример создания :ref:`шаблона нумерации<sample_number_template>`
      * - :ref:`Журналы<journals>`
        - Например, журнал, сгенерированный автоматически на основании созданного :ref:`типа данных "Заявка на покупку оборудования"<ws_artifacts_data_type>`.
      * - :ref:`Формы<forms>`
        - Например, форма, сгенерированная автоматически на основании созданного :ref:`типа данных "Заявка на покупку оборудования"<ws_artifacts_data_type>`.
      * - :ref:`Меню<menu>`
        - Сохраненные конфигурации :ref:`меню<menu_ws_settings>`
      * - :ref:`Дашборды<dashboard>`
        - Созданные :ref:`дашборды<sample_dashboard>`
      * - :ref:`Канбан-доски<kanban_board>`
        - Созданные :ref:`канбан-доски<sample_kanban_board>`
      * - :ref:`Модели BPMN<ecos_bpmn>`
        - Например, созданная :ref:`Модель BPMN "Заявка на покупку оборудования"<ws_artifacts_bpmn>`.
      * - :ref:`Модели DMN<ecos_bpmn>`
        - Созданные DMN решения. Пример включения :ref:`таблицы принятия решений<sample_request_dmn>` в бизнес-процесс.
      * - :ref:`Шаблоны уведомлений<notification_templates>`
        - Созданные :ref:`шаблоны уведомлений<notification_templates>`
      * - :ref:`Уведомления<notifications-label>`
        - Отправленные уведомления

См. подробно пример создания :ref:`Заявки на покупку оборудования<ws_artifacts_data_type>`.
