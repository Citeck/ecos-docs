------------------------------------------------------------------------

Пространство предназначено для системных администраторов и содержит инструменты конфигурации и настройки платформы.

Конфигурации рабочего пространства
------------------------------------------------------------------------

Рабочее пространство, которое должно открываться по умолчанию
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Для пользователей по умолчанию открывается :ref:`персональное рабочее пространство<ws_personal>`. Если необходимо, чтобы все пользователи сразу попадали в определённое пространство (например, на корпоративный портал), это можно настроить через конфигурацию **default-workspace**.

Перейдите в журнал :ref:`«Конфигурация ECOS»<configuration_admin>`. Журнал доступен по адресу: ``v2/journals?journalId=ecos-configs&viewMode=table&ws=admin$workspace``

Конфигурация **default-workspace**:

 .. image:: _static/default_setting_01.png
       :width: 600
       :align: center

Укажите **id** рабочего пространства, которое должно открываться по умолчанию:

 .. image:: _static/default_setting_02.png
       :width: 400
       :align: center

Включить/Выключить возможность создания рабочих пространств всем пользователям
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Конфигурация **workspaces-allow-create-for-all-users** управляет доступностью кнопки создания рабочих пространств для рядовых пользователей. Например, на внешнем портале эту возможность можно отключить за ненадобностью.

 .. image:: _static/allow_ws_01.png
       :width: 600
       :align: center

По умолчанию настройка включена — создание пространств доступно всем пользователям.

 .. image:: _static/allow_ws_02.png
       :width: 400
       :align: center
