.. _content_service:

Content микросервис
=========================

.. note::

    Доступно только в Enterprise версии.

**Ecos-content** - микросервис, предназначен для обеспечения хранения файлов в системе в определенное файловое хранилище. Микросервис позволяет настроить конфигурацию системы таким образом, чтобы ECOS имел возможность взаимодействовать одновременно с несколькими файловыми хранилищами и обеспечивать возможность сохранения разных типов объектов системы в разные хранилища.
