Предпросмотр документов
=======================

Функционал предпросмотра позволяет отображать содержимое документов различных
форматов (PDF, изображения, офисные документы и др.) непосредственно
в веб-интерфейсе Citeck без необходимости скачивания файла. Для преобразования
документов в формат, пригодный для отображения в браузере, используется сервис
трансформации контента. В данном разделе описаны настройка предпросмотра
документов и конфигурация трансформации контента.
