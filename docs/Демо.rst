Demo стенд
==================

Демо-стенд Citeck — это предварительно настроенная среда, позволяющая ознакомиться
с основными возможностями платформы без необходимости локальной установки.
На демо-стенде доступны примеры бизнес-процессов, типовые журналы, формы
и дашборды. Стенд предназначен для первичного знакомства с системой, тестирования
сценариев автоматизации и оценки функциональности перед принятием решения
о внедрении.

Доступ к демо-стенду можно получить, заполнив заявку на странице
https://www.citeck.ru/community/download-community-version/
