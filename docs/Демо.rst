Демо-стенд
==========

Демо-стенд Citeck — предварительно настроенная среда для знакомства с основными возможностями платформы без необходимости локальной установки.

Стенд предназначен для:

- первичного знакомства с системой;
- тестирования сценариев автоматизации;
- оценки функциональности перед принятием решения о внедрении.

На демо-стенде доступны примеры бизнес-процессов, типовые журналы, формы и дашборды.

Получение доступа
------------------

Для получения доступа к демо-стенду заполните заявку на сайте Citeck:
`Страница загрузки Community Edition <https://www.citeck.ru/community/download-community-version/>`_
