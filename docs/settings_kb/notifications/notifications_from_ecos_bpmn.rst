Отправка уведомлений из Citeck BPMN (Camunda)
===============================================

Платформа Citeck позволяет настраивать автоматическую отправку уведомлений
непосредственно из BPMN-процессов, построенных на базе Camunda. Для этого
используется специальный компонент отправки уведомлений, который может быть
встроен в любой бизнес-процесс в виде задачи (Service Task). Уведомления
поддерживают шаблоны и динамическую подстановку данных из контекста процесса.
Конфигурация выполняется через редактор BPMN-процессов в интерфейсе Citeck.

См. описание компонента :ref:`Уведомлений<notification>`
