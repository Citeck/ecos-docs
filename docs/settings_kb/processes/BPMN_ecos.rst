Платформа бизнес-процессов (Citeck BPMN)
========================================

.. _ecos-bpmn_platform:

В Citeck реализована low-code платформа для автоматизации бизнес-процессов, которая использует нотацию BPMN (Business Process Model and Notation) для моделирования и выполнения процессов. Встроенный визуальный BPMN-редактор позволяет создавать схемы процессов и оптимизировать их для автоматизации.
